# Contributing

Simply clone the repo:
```
https://github.com/bitgrin/www.bitgrin.dev.git
```

Then run ```serve.bat``` (requires python)
This will start a local webserver at ```http://127.0.0.1:8000```

Make a fork on Github and open a PR to contribute changes.
